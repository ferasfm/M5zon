───────────────────────────────────────────────────────────────
🚀 الحل السريع (3 خطوات)
───────────────────────────────────────────────────────────────

الخطوة 1: افتح Supabase SQL Editor
   → https://supabase.com
   → افتح مشروعك
   → SQL Editor

الخطوة 2: شغّل هذا الأمر للفحص
   ┌─────────────────────────────────────────────────┐
   │ SELECT                                          │
   │     p.name as المحافظة,                         │
   │     a.name as المنطقة                           │
   │ FROM areas a                                    │
   │ JOIN provinces p ON a.province_id = p.id        │
   │ ORDER BY p.name, a.name;                        │
   └─────────────────────────────────────────────────┘

الخطوة 3: صحح البيانات الخاطئة
   → راجع النتائج
   → إذا وجدت منطقة في محافظة خاطئة، صححها

───────────────────────────────────────────────────────────────
📝 مثال على التصحيح
───────────────────────────────────────────────────────────────

إذا وجدت:
   ❌ المحافظة: رام الله
      المنطقة: نابلس ← خطأ!

الحل:
   ┌─────────────────────────────────────────────────┐
   │ UPDATE areas                                    │
   │ SET province_id = (                             │
   │     SELECT id FROM provinces WHERE name='نابلس' │
   │ )                                               │
   │ WHERE name = 'نابلس';                           │
   └─────────────────────────────────────────────────┘
