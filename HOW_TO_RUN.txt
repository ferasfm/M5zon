═══════════════════════════════════════════════════════════════
🎯 كيفية التشغيل
═══════════════════════════════════════════════════════════════

الطريقة الأولى (الأسهل):
─────────────────────
1. انقر مرتين على: start-electron.bat
2. انتظر قليلاً حتى يفتح التطبيق
3. استمتع باستخدام التطبيق!

الطريقة الثانية (سطر الأوامر):
──────────────────────────────
1. افتح Command Prompt أو PowerShell
2. اكتب: npm run dev:electron
3. اضغط Enter

═══════════════════════════════════════════════════════════════
📦 كيفية بناء التطبيق للإنتاج
═══════════════════════════════════════════════════════════════

الطريقة الأولى (الأسهل):
─────────────────────
1. انقر مرتين على: build-electron.bat
2. انتظر حتى ينتهي البناء
3. ستجد ملفات التثبيت في مجلد: dist/

الطريقة الثانية (سطر الأوامر):
──────────────────────────────
1. افتح Command Prompt أو PowerShell
2. اكتب: npm run build:electron
3. اضغط Enter

═══════════════════════════════════════════════════════════════
🛑 كيفية إيقاف التطبيق
═══════════════════════════════════════════════════════════════

الطريقة الأولى:
──────────────
1. انقر مرتين على: stop-electron.bat

الطريقة الثانية:
───────────────
1. أغلق نافذة التطبيق مباشرة

الطريقة الثالثة:
───────────────
1. اضغط Ctrl+C في سطر الأوامر
