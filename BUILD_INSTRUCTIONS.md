# تعليمات بناء النسخة النهائية

## المتطلبات
- Node.js v18 أو أحدث
- npm

## خطوات البناء

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. بناء النسخة النهائية
```bash
npm run build:electron
```

### 3. الملفات الناتجة
بعد اكتمال البناء، ستجد الملفات في مجلد `dist`:

- **إدارة المخزون الاحترافية 1.0.0.exe** - نسخة Portable (لا تحتاج تثبيت)
- **إدارة المخزون الاحترافية Setup 1.0.0.exe** - برنامج التثبيت (NSIS Installer)

## أنواع النسخ

### Portable Version
- لا تحتاج تثبيت
- يمكن تشغيلها من أي مكان
- يمكن نسخها على USB
- الإعدادات تُحفظ في نفس المجلد

### Installer Version (NSIS)
- برنامج تثبيت احترافي
- يضيف اختصارات في سطح المكتب وقائمة ابدأ
- يسجل البرنامج في Windows
- يوفر خيار إلغاء التثبيت

## إضافة أيقونة مخصصة

1. ضع ملف الأيقونة (.ico) في مجلد `public` باسم `icon.ico`
2. تأكد من أن الأيقونة بحجم 256x256 بكسل على الأقل
3. أعد بناء البرنامج

## حل المشاكل

### مشكلة: "Permission denied" أثناء البناء
**الحل:** أغلق البرنامج إذا كان مفتوحاً، ثم نفذ:
```bash
taskkill /F /IM "إدارة المخزون الاحترافية.exe"
npm run build:electron
```

### مشكلة: صفحة بيضاء عند التشغيل
**الحل:** تأكد من أن `base: './'` موجود في `vite.config.ts`

### مشكلة: الأيقونة الافتراضية تظهر
**الحل:** أضف ملف `icon.ico` في مجلد `public` وأعد البناء
