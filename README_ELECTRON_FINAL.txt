📋 الأوامر المتاحة
═══════════════════════════════════════════════════════════════

تطوير:
  npm run dev              - تشغيل الويب (القديم)
  npm run dev:electron     - تشغيل Electron (جديد)
  npm run dev:vite         - تشغيل خادم Vite فقط

بناء:
  npm run build            - بناء الويب (القديم)
  npm run build:electron   - بناء Electron (جديد)

أخرى:
  npm run preview          - معاينة البناء
  npm run status           - فحص الحالة
  npm run stop             - إيقاف التطبيق
