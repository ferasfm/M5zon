═══════════════════════════════════════════════════════════════
📍 الخطوة 1: التحقق من المتطلبات
═══════════════════════════════════════════════════════════════

ماذا تحتاج:
  ✓ Windows 7 أو أحدث
  ✓ Node.js (v14 أو أحدث)
  ✓ اتصال بالإنترنت (للتشغيل الأول فقط)

كيفية التحقق من Node.js:

  الخطوة 1: افتح Command Prompt
    ➤ اضغط: Windows + R
    ➤ اكتب: cmd
    ➤ اضغط: Enter

  الخطوة 2: تحقق من Node.js
    ➤ اكتب: node --version
    ➤ اضغط: Enter
    ➤ يجب أن تظهر نسخة مثل: v18.0.0

  إذا لم تظهر نسخة:
    ➤ حمّل Node.js من: https://nodejs.org
    ➤ اختر الإصدار LTS
    ➤ ثبّت البرنامج
    ➤ أعد تشغيل الكمبيوتر

═══════════════════════════════════════════════════════════════
📍 الخطوة 2: التشغيل الأول (الطريقة الأسهل)
═══════════════════════════════════════════════════════════════

ببساطة:

  الخطوة 1: ابحث عن الملف
    ➤ افتح مجلد المشروع
    ➤ ابحث عن: start-electron.bat

  الخطوة 2: شغّل الملف
    ➤ انقر مرتين على: start-electron.bat
    ➤ ستظهر نافذة سوداء (Command Prompt)

  الخطوة 3: انتظر
    ➤ انتظر 30-60 ثانية
    ➤ ستظهر رسائل مثل:
      - "Node.js مثبت"
      - "التبعيات موجودة"
      - "بدء تطبيق Electron"

  الخطوة 4: استمتع
    ➤ سيفتح البرنامج تلقائياً
    ➤ ستظهر نافذة البرنامج
    ➤ البرنامج جاهز للاستخدام

═══════════════════════════════════════════════════════════════
📍 الخطوة 3: استخدام البرنامج
═══════════════════════════════════════════════════════════════

الآن يمكنك:
  ✓ استخدام جميع الوظائف
  ✓ إدارة المخزون
  ✓ إضافة المنتجات
  ✓ إدارة الموردين
  ✓ عرض التقارير

═══════════════════════════════════════════════════════════════
📍 الخطوة 4: إيقاف البرنامج
═══════════════════════════════════════════════════════════════

عند الانتهاء:

  الطريقة 1 (الأسهل):
    ➤ انقر مرتين على: stop-electron.bat

  الطريقة 2 (من البرنامج):
    ➤ أغلق نافذة البرنامج بالطريقة العادية

  الطريقة 3 (من Command Prompt):
    ➤ افتح Command Prompt
    ➤ اكتب: npm run stop
    ➤ اضغط: Enter

📍 الخطوة 5: التشغيل اللاحق
═══════════════════════════════════════════════════════════════

في المرات القادمة:

  ببساطة:
    ➤ انقر مرتين على: start-electron.bat
    ➤ انتظر 5-10 ثوان
    ➤ البرنامج يفتح

═══════════════════════════════════════════════════════════════
📍 الخطوة 6: بناء البرنامج (اختياري)
═══════════════════════════════════════════════════════════════

إذا أردت إنشاء ملف تثبيت:

  الخطوة 1: ابحث عن الملف
    ➤ افتح مجلد المشروع
    ➤ ابحث عن: build-electron.bat

  الخطوة 2: شغّل الملف
    ➤ انقر مرتين على: build-electron.bat
    ➤ ستظهر نافذة سوداء

  الخطوة 3: انتظر
    ➤ انتظر 2-5 دقائق
    ➤ ستظهر رسالة: "تم بناء التطبيق بنجاح"

  الخطوة 4: ابحث عن ملفات التثبيت
    ➤ افتح مجلد: dist/
    ➤ ستجد ملفات التثبيت

═══════════════════════════════════════════════════════════════
📊 ملخص الأوامر
═══════════════════════════════════════════════════════════════

للمستخدم العادي:
  [1] انقر مرتين على: start-electron.bat
  [2] استخدم البرنامج
  [3] انقر مرتين على: stop-electron.bat

للمطور (Command Prompt):
  [1] npm install              - تثبيت المكتبات
  [2] npm run dev:electron     - تشغيل البرنامج
  [3] npm run build:electron   - بناء البرنامج
  [4] npm run stop             - إيقاف البرنامج
