📋 الخطوات بالترتيب (للمستخدم الجديد)
═══════════════════════════════════════════════════════════════

إذا كان البرنامج جديداً ولم يتم تشغيله نهائياً، اتبع هذه الخطوات:

═══════════════════════════════════════════════════════════════
الخطوة 1️⃣: التحقق من Node.js
═══════════════════════════════════════════════════════════════

تأكد من تثبيت Node.js على جهازك:

الطريقة الأولى (سطر الأوامر):
  1. افتح Command Prompt أو PowerShell
  2. اكتب: node --version
  3. يجب أن تظهر نسخة مثل: v18.0.0 أو أعلى

إذا لم يظهر شيء:
  ➤ حمّل Node.js من: https://nodejs.org
  ➤ اختر الإصدار LTS (الموصى به)
  ➤ ثبّت البرنامج بالطريقة العادية
  ➤ أعد تشغيل الكمبيوتر

═══════════════════════════════════════════════════════════════
الخطوة 2️⃣: التحقق من npm
═══════════════════════════════════════════════════════════════

تأكد من تثبيت npm (يأتي مع Node.js):

الطريقة الأولى (سطر الأوامر):
  1. افتح Command Prompt أو PowerShell
  2. اكتب: npm --version
  3. يجب أن تظهر نسخة مثل: 9.0.0 أو أعلى

إذا لم يظهر شيء:
  ➤ أعد تثبيت Node.js
  ➤ أعد تشغيل الكمبيوتر

═══════════════════════════════════════════════════════════════
الخطوة 3️⃣: الطريقة الأسهل (للمستخدم العادي)
═══════════════════════════════════════════════════════════════

ببساطة:
  1. انقر مرتين على: start-electron.bat
  2. انتظر قليلاً (قد يستغرق 30-60 ثانية)
  3. سيتم تثبيت جميع المكتبات تلقائياً
  4. سيفتح البرنامج تلقائياً

هذا كل شيء! ✅

═══════════════════════════════════════════════════════════════
الخطوة 4️⃣: الطريقة اليدوية (للمطورين)
═══════════════════════════════════════════════════════════════

إذا أردت التحكم الكامل:

الخطوة 1: فتح سطر الأوامر
  ➤ افتح Command Prompt أو PowerShell
  ➤ انتقل إلى مجلد المشروع

الخطوة 2: تثبيت المكتبات
  اكتب: npm install
  انتظر حتى ينتهي (قد يستغرق 5-10 دقائق)

الخطوة 3: تشغيل البرنامج
  اكتب: npm run dev:electron
  انتظر حتى يفتح البرنامج

═══════════════════════════════════════════════════════════════
الخطوة 5️⃣: بناء البرنامج (للتوزيع)
═══════════════════════════════════════════════════════════════

إذا أردت إنشاء ملف تثبيت:

الطريقة الأسهل:
  1. انقر مرتين على: build-electron.bat
  2. انتظر حتى ينتهي البناء
  3. ستجد ملفات التثبيت في مجلد: dist/

الطريقة اليدوية:
  اكتب: npm run build:electron
  انتظر حتى ينتهي

═══════════════════════════════════════════════════════════════
الخطوة 6️⃣: إيقاف البرنامج
═══════════════════════════════════════════════════════════════

الطريقة الأسهل:
  1. انقر مرتين على: stop-electron.bat

الطريقة اليدوية:
  اكتب: npm run stop

═══════════════════════════════════════════════════════════════
📊 ملخص الأوامر الأساسية
═══════════════════════════════════════════════════════════════

للمستخدم العادي:
  [1] انقر مرتين على: start-electron.bat
  [2] استخدم البرنامج
  [3] انقر مرتين على: stop-electron.bat

للمطور:
  [1] npm install              - تثبيت المكتبات
  [2] npm run dev:electron     - تشغيل البرنامج
  [3] npm run build:electron   - بناء البرنامج
  [4] npm run stop             - إيقاف البرنامج

═══════════════════════════════════════════════════════════════
❓ أسئلة شائعة
═══════════════════════════════════════════════════════════════

س: كم من الوقت يستغرق التشغيل الأول؟
ج: 30-60 ثانية (لتثبيت المكتبات)

س: هل أحتاج إلى إنترنت؟
ج: نعم، للتشغيل الأول فقط (لتحميل المكتبات)

س: ماذا لو حدث خطأ؟
ج: اقرأ الرسالة بعناية وتحقق من:
   - تثبيت Node.js
   - الاتصال بالإنترنت
   - توفر المنفذ 5173

س: هل يمكنني نقل البرنامج إلى جهاز آخر؟
ج: نعم، انسخ المجلد كاملاً وشغّل start-electron.bat

س: كيف أحدّث البرنامج؟
ج: اسحب أحدث نسخة من المشروع وشغّل start-electron.bat
